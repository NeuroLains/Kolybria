Проект React + Vite для каталога полиграфии и сувенирной продукции
============================================================

1. Основная структура папок:
---------------------------
- src/
  - assets/ — изображения товаров, иконки, галереи, ассеты для разных категорий (polygraphy, souvenirs, badges, blanks и др.)
  - components/ — переиспользуемые компоненты (ProductCard, OrderModal и др.)
  - data/ — централизованные данные о товарах (products.js)
  - pages/ — страницы сайта (Home, ProductPage, Polygraphy, Souvenirs, Services, Contacts и др.)
  - utils/ — утилиты (например, priceFormatter.js для форматирования цен)
  - App.jsx — корневой компонент с маршрутизацией
  - main.jsx — точка входа

2. Маршрутизация и страницы:
---------------------------
- Реализована через react-router-dom (BrowserRouter, Routes, Route).
- Основные страницы:
  - / (Home) — главная, слайдер, галерея, лучшие товары
  - /polygraphy — полиграфия (визитки, листовки, буклеты и др.)
  - /souvenirs — сувенирная продукция (значки, кружки, футболки и др.)
  - /services — услуги (дизайн, печать и др.)
  - /contacts — контакты
  - /product/:id — страница отдельного товара с калькулятором опций и заказом
  - /advert, /about, /blanks, /badges, /printing, /stands, /copycenter и др. — дополнительные страницы

3. Данные о товарах:
--------------------
- src/data/products.js — массив объектов товаров с полями:
  - id, name, image, carouselImages, description, basePrice, options (различные опции: таблицы, radio, checkbox и др.)
  - Для каждого товара могут быть разные опции (тираж, материал, размер и др.)
- Изображения товаров импортируются из assets/services/*

4. Карточки товаров:
--------------------
- Компонент ProductCard (src/components/ProductCard.jsx):
  - Принимает props: to (ссылка), image, title, price
  - Используется на страницах Polygraphy, Souvenirs, Services и др. для отображения сетки товаров
  - Стилизация через ProductCard.css

5. Страница товара (ProductPage):
---------------------------------
- src/pages/ProductPage.jsx
- Использует useParams для получения id товара
- Находит товар в products по id
- Отображает галерею, описание, калькулятор опций (например, для стикеров — пошаговый выбор формы, размера, тиража, материала)
- Калькулятор реализован как отдельная функция-компонент внутри ProductPage
- Есть OrderModal для оформления заказа (отправка данных на сервер через fetch)

6. Форматирование цен:
---------------------
- src/utils/priceFormatter.js — функция getFormattedPrice(product)
  - Вычисляет минимальную цену за единицу для карточки товара, учитывая basePrice, таблицы опций, модификаторы и др.

7. Ассеты и иконки:
-------------------
- src/assets/icons — svg-иконки для категорий и товаров
- src/assets/services — изображения для разных категорий (polygraphy, souvenirs и др.)
- src/assets/gallery/Cards — изображения для галереи и карточек

8. Прочее:
----------
- В проекте есть стили для компонентов и страниц (App.css, ProductCard.css и др.)
- README.md содержит только базовую информацию о шаблоне Vite + React

Итого: проект реализует каталог с фильтрацией по категориям, подробными карточками товаров, калькулятором опций и возможностью заказа. Все данные о товарах централизованы в products.js, карточки и страницы построены на переиспользуемых компонентах, маршрутизация через react-router-dom.
